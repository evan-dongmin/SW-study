# 도커 + 프로메테우스 + 그라파나로 모니터링 시스템 구축하기
# ================================================

1. docker-copmose.yml 파일 작성
2. prometheus.yml 파일 작성
3. docker-compose로 컨테이너 실행
4. 그라파나 대시보드 설정
   - https://grafana.com/grafana/dashboards/ 에서 대시보드 템플릿을 가져올 수 있음. (Spring Boot APM Temp,JVM Temp, Hikari/JDBC Temp 등)
5. 모니터링 결과 확인
